- `full_loss`: 完整损失（推荐），平衡拟人性和精度

## 指标说明

| 指标 | 说明 | 单位 |
|------|------|------|
| Params (K) | 模型参数量 | 千 |
| Network (ms) | 神经网络推理延迟 | 毫秒 |
| IK (ms) | IK 求解延迟 | 毫秒 |
| Total (ms) | 总延迟 (Network + IK) | 毫秒 |
| Swivel MAE | 臂角平均绝对误差 | 度 |
| Elbow Error | 肘部位置误差 | 毫米 |
| Jerk | 动作平滑度（越小越平滑） | - |
