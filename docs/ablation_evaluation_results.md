## 指标说明

| 指标 | 说明 | 单位 |
|------|------|------|

| IK (ms) | IK 求解延迟 | 毫秒 |

| Elbow Error | 肘部位置误差 | 毫米 |
| Jerk | 动作平滑度（越小越平滑） | - |
